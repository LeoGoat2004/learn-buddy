## Quick Start

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/learn-buddy.git
cd learn-buddy

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

---

## Usage Guide

### 1. Add Knowledge

1. Click "Add Knowledge" in the sidebar
2. Enter a URL and click "Auto Fetch" to extract content
3. Fill in title, select/create topic, add notes
4. Click "Save Knowledge"

### 2. Review with AI

1. Configure AI in Settings (optional)
2. Go to Review page
3. Ask questions or let AI quiz you

### 3. Switch Language

Click the 🌐 button in the header to toggle between Chinese and English.
